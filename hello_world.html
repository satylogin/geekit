HELLO WORLD!

Let us write our first program in C language. The problem is the classic hello world problem.

#include

int main()
{
printf("HELLO WORLD!\n");

return 0;
}

To Execute

Go on and write the above code in the editor. compile and run the code.
For linux users, type ctrl + alt + t to open the terminal. Then type vi hello_world.c.
This will open up a new file called hello_world.c
hello_world is the name of the file and .c is the extension that tells the compiler that this file is a c program source code.
press i to start typing, i stands for insert mode. type the following code in the file. to save and exit, press esc. Then type ":x" and then enter. Now you have saved and exit.
To compile the code, type gcc hello_world.c. This will compile the code.
Now the final step, type ./a.out to run your program.

Let us Disect

#include :
This means that we are telling the compiler to include a header file called stdio.h (.h means header) which has functions like scanf and printf that helps us to take input and give output. We will include a seperate article on #include.
int main() { } :
every c function must have a main function. This is the place from where the file starts to execute. int means the return type of the function. int is a data type. We will add another article on data types in c. :
return 0; :
This is the return value of the function. It means what the function returns to the calling function.
For now just memorise that these will always be in your c program, no matter whatever program you are making.

And finally printf("HELLO WORLD\n") :
It says to print the string HELLO WORLD on screen.
\n Stands for new line.
It will be easy to write code if you memorise these steps. With time as you will read further, you will understand these things in more detail. And it gives you a little bit of confidence if you are able to write code.
